# NEXUS: AI-Driven Mentor-Mentee Ecosystem

NEXUS is a high-fidelity, professional mentorship platform designed to facilitate seamless collaboration between mentors and mentees. Built with a cutting-edge serverless architecture, it provides robust scheduling, real-time communication, and automated progress tracking.

## 🚀 Key Features

- **Unified Auth Engine**: Secure Magic Link and password-based authentication via Supabase GoTrue.
- **Intelligent Scheduler**: Conflict-free session booking with automated status tracking.
- **Synergy Messaging**: Real-time WebSocket-driven communication bridge.
- **Progressive Task Tracker**: Automated heuristic evaluation of mentee progress.
- **Asset Repository**: Secure, centralized library for scholarship and assignment materials.

## 🛠 Tech Stack

- **Frontend**: [Next.js 16+](https://nextjs.org) (App Router), [React 19+](https://react.dev)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com), [Framer Motion](https://www.framer.com/motion/)
- **Backend**: [Supabase](https://supabase.com) (PostgreSQL, Realtime, Auth, Storage)
- **Infrastructure**: [Firebase](https://firebase.google.com) (Hosting & Frameworks)

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Harshx1211/nexus.git
   cd nexus
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment Variables**:
   Create a `.env.local` file with your Supabase credentials.

4. **Launch Development Server**:
   ```bash
   npm run dev
   ```
